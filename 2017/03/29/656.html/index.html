Ubuntu 16.04 root用户耳机木有声音的解决办法

2017-03-29 02:50:56
阅读约 1 分钟
122 字
标签 Ubuntu
分类 问题

今天把Ubuntu默认的用户换成root用户以后发现会出现木有声音的情况，百度了下找了个简单点的解决办法

执行：pulseaudio --start --log-target=syslog

接着应该就能听见声音了，如果还是木有，自行百度吧～

添加开机启动

左上角的齿轮里打开启动应用程序，添加这段命令即可
