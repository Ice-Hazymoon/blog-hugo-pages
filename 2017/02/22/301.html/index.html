很早之前就想用树莓派当闹钟了，尝试过许多方法，比如用谷歌日历，PHP脚本等，但最后尝试下来效果都不是很满意。今天在知乎上看到了一篇用python脚本当闹钟的方法，尝试了一下，霸气的不要不要的，自带天气播放功能~所以就顺便写个教程吧

准备

  • 树莓派*1
  • 音箱*1

环境

  • Python3.4

安装依赖

  • #树莓派自带的交易版本是2.7的，所以到最后不能正常运行，我在这里折腾了很久。。。首先把py的版本改成3.4~
    sudo ln -s /usr/bin/python3.4 /usr/bin/python #把系统默认python命令改成python3
  • #从 https://pypi.python.org/pypi/pip#downloads 下载最新版本的pip
    #把刚刚下载的文件上传后解压
    tar -xzvf pip-9.0.1.tar.gz
    #进入安装目录
    cd pip-9.0.1
    #进行安装
    python setup.py install
  • #安装Beautiful Soup4
    #可通过系统的软件包管理来安装
    apt-get install Python-bs4
    #也可以用easy_install安装
    easy_install beautifulsoup4
    #也可以用pip进行安装（推荐）
    pip install beautifulsoup4
  • #安装Requests
    #通过pip安装（推荐）
    pip install requests
    #也可以通过git库安装
    git clone git://github.com/kennethreitz/requests.git
    cd requests
    python setup.py install

文字转语音

使用百度的文字转换语音API，地址：百度语音-永久免费智能语音开放平台 还可以选各种声音，调节语速。虽然它没有给出直接的api接口，但是我们利用Chrome浏览器的开发者模式可以找到api。 打开开发者模式，点击播放的按钮，在network里就可以找到刚刚发出的请求。

其中 http://tts.baidu.com/text2audio?idx=1&tex=1&cuid=baidu_speech_demo&cod=2&lan=zh&ctp=1&pdt=1&spd=5&per=4&vol=5&pit=5 就是我们要找的百度文字转语音API,其中per是参数是语音的类型，spd是语速，vol是音量，而tex则是需要转换的文字。通过以下代码就可以实现将特定的文字转换为语音，并以mp3格式保存到本地。

实现定时播放语音

使用mplayer实现语音播放,通过以下命令安装mplayer:

sudo apt-get install mplayer
用法很简单输入以下命令即可播放音乐：

mplayer \xxx\xxx\xxx.mp3(绝对地址)

本来是用crontab来实现定时播放的，但是后来发现了一个bug。程序运行的流程是实时下载语音MP3文件到本地，然后用os.system()直接调用mplayer播放语音。程序本地运行时正常，用crontab定时运行就找不到实时下载的语音文件。没有办法就自己写了一个每天定时播放的脚本。

 1import time
 2from datetime import datetime
 3
 4def get_seconds(h='07', m='30', s='00'):
 5    """获取当前时间与程序启动时间间隔秒数"""
 6
 7    # 设置程序启动的时分秒
 8    time_pre = '%s:%s:%s' % (h, m, s)
 9    # 获取当前时间
10    time1 = datetime.now()
11    # 获取程序今天启动的时间的字符串格式
12    time2 = time1.date().strftime('%Y-%m-%d') + ' ' + time_pre
13    # 转换为datetime格式
14    time2 = datetime.strptime(time2, '%Y-%m-%d %H:%M:%S')
15    # 判断当前时间是否晚于程序今天启动时间，若晚于则程序启动时间增加一天
16    if time1 > time2:
17        time2 = time2 + timedelta(days=1)
18
19    return time.mktime(time2.timetuple()) - time.mktime(time1.timetuple())

该函数默认计算当前事件距上午七点半间隔秒数，需要修改天气播报事件就自己修改三个默认参数，h是小时，m是分钟，s是秒。

 14headers = {

 21def numtozh(num):
 22    num_dict = {1: '一', 2: '二', 3: '三', 4: '四', 5: '五', 6: '六', 7: '七',
 23                8: '八', 9: '九', 0: '零'}
 24    num = int(num)
 25    if 100 <= num < 1000:
 26        b_num = num // 100
 27        s_num = (num-b_num*100) // 10
 28        g_num = (num-b_num*100) % 10
 29        if g_num == 0 and s_num == 0:
 30            num = '%s百' % (num_dict[b_num])
 31        elif s_num == 0:
 32            num = '%s百%s%s' % (num_dict[b_num], num_dict.get(s_num, ''), num_dict.get(g_num, ''))
 33        elif g_num == 0:
 34            num = '%s百%s十' % (num_dict[b_num], num_dict.get(s_num, ''))
 35        else:
 36            num = '%s百%s十%s' % (num_dict[b_num], num_dict.get(s_num, ''), num_dict.get(g_num, ''))
 37    elif 10 <= num < 100:
 38        s_num = num // 10
 39        g_num = (num-s_num*10) % 10
 40        if g_num == 0:
 41            g_num = ''
 42        num = '%s十%s' % (num_dict[s_num], num_dict.get(g_num, ''))
 43    elif 0 <= num < 10:
 44        g_num = num
 45        num = '%s' % (num_dict[g_num])
 46    elif -10 < num < 0:
 47        g_num = -num
 48        num = '零下%s' % (num_dict[g_num])
 49    elif -100 < num <= -10:
 50        num = -num
 51        s_num = num // 10
 52        g_num = (num-s_num*10) % 10
 53        if g_num == 0:
 54            g_num = ''
 55        num = '零下%s十%s' % (num_dict[s_num], num_dict.get(g_num, ''))
 56    return num
 57

 59def get_seconds(h='07', m='30', s='00'):
 60    """获取当前时间与程序启动时间间隔秒数"""
 61
 62    # 设置程序启动的时分秒
 63    time_pre = '%s:%s:%s' % (h, m, s)
 64    # 获取当前时间
 65    time1 = datetime.now()
 66    # 获取程序今天启动的时间的字符串格式
 67    time2 = time1.date().strftime('%Y-%m-%d') + ' ' + time_pre
 68    # 转换为datetime格式
 69    time2 = datetime.strptime(time2, '%Y-%m-%d %H:%M:%S')
 70    # 判断当前时间是否晚于程序今天启动时间，若晚于则程序启动时间增加一天
 71    if time1 > time2:
 72        time2 = time2 + timedelta(days=1)
 73
 74    return time.mktime(time2.timetuple()) - time.mktime(time1.timetuple())

 77def get_weather():
 78    # 下载墨迹天气主页源码
 79    res = requests.get('http://tianqi.moji.com/', headers=headers)
 80    # 用BeautifulSoup获取所需信息
 81    soup = BeautifulSoup(res.text, "html.parser")
 82    temp = soup.find('div', attrs={'class': 'wea_weather clearfix'}).em.getText()
 83    temp = numtozh(int(temp))
 84    weather = soup.find('div', attrs={'class': 'wea_weather clearfix'}).b.getText()
 85    sd = soup.find('div', attrs={'class': 'wea_about clearfix'}).span.getText()
 86    sd_num = re.search(r'\d+', sd).group()
 87    sd_num_zh = numtozh(int(sd_num))
 88    sd = sd.replace(sd_num, sd_num_zh)
 89    wind = soup.find('div', attrs={'class': 'wea_about clearfix'}).em.getText()
 90    aqi = soup.find('div', attrs={'class': 'wea_alert clearfix'}).em.getText()
 91    aqi_num = re.search(r'\d+', aqi).group()
 92    aqi_num_zh = numtozh(int(aqi_num))
 93    aqi = aqi.replace(aqi_num, aqi_num_zh).replace(' ', ',空气质量')
 94    info = soup.find('div', attrs={'class': 'wea_tips clearfix'}).em.getText()
 95    sd = sd.replace(' ', '百分之').replace('%', '')
 96    aqi = 'aqi' + aqi
 97    info = info.replace('，', ',')
 98    # 获取今天的日期
 99    today = datetime.now().date().strftime('%Y年%m月%d日')
100    # 将获取的信息拼接成一句话
101    text = '早上好！今天是%s,天气%s,温度%s摄氏度,%s,%s,%s,%s' % \
102            (today, weather, temp, sd, wind, aqi, info)
103    return text

106def text2voice(text):
107    url = 'http://tts.baidu.com/text2audio?idx=1&tex={0}&cuid=baidu_speech_' \
108            'demo&cod=2&lan=zh&ctp=1&pdt=1&spd=4&per=4&vol=5&pit=5'.format(text)
109    # 下载转换后的mp3格式语音
110    res = requests.get(url, headers=headers)
111    # 将MP3存入本地
112    with open('1.mp3', 'wb') as f:
113        f.write(res.content)

116def main():
117    while True:
118        s = get_seconds()
119        time.sleep(s)
120        # 获取需要转换语音的文字
121        text = get_weather()
122        print(text)
123        # 将文字转换为语音并存入程序所在文件夹
124        text2voice(text)
125        # 获取音乐文件绝对地址
126        mp3path2 = os.path.join(os.path.dirname(__file__), '2.mp3')
127        # 先播放一首音乐做闹钟
128        os.system('mplayer %s' % mp3path2)
129        # 播报语音天气
130        mp3path1 = os.path.join(os.path.dirname(__file__), '1.mp3')
131        os.system('mplayer %s' % mp3path1)
132        os.remove(mp3path1)
