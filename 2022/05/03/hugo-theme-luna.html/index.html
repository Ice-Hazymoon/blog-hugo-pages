一个简洁，快速，SEO友好的 HUGO 主题

鸽了大半年，终于🕊上线了，这是一个废了不少头发才开发出来的主题，集性能、美观、SEO于一体的主题，并且支持文章加密和 GitHub Actions 自动部署，你只需要安心写博客就行。

开发的过程中~参考~了不少主题和代码，求各位大佬来个🌟吧~

Ice-Hazymoon/hugo-theme-luna
A simple, performance-first, SEO-friendly Hugo theme / 一个轻量，快速，SEO 友好的 Hugo 主题
JavaScript
93
7

特性

  • 使用 Tailwindcss 构建
  • 大量短代码支持，点击查看
  • 按需引入 JS 模块
  • 数学公式支持
  • PWA 支持，可以直接添加到桌面
  • PJAX（页面无刷新加载）
  • 自定义主题颜色和字体
  • 支持 Bionic Reading ，更快的阅读英文文章
  • 夜间模式
  • 本地搜索支持
  • 文章目录（TOC）
  • 图片渐进式加载和懒加载
  • 文章加密功能
  • 多语言支持，同时适配谷歌翻译
  • <noscript> 支持，即使不启用 JS 也能正常使用大部分功能
  • 支持画廊页面，也可以用于文章页面
  • 自动生成 Twitter 页面
  • 自动生成 GitHub 页面
  • GitHub Actions 支持，自动部署你的博客到 GitHub Pages
  • 等等…..
