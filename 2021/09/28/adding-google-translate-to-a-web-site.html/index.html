网站懒得做 i18n？

不如直接谷歌翻译

特性

  • 使用 translate.google.cn 资源，大陆地区也能使用
  • 自动判断浏览器语言
  • 支持 Google 翻译支持的所有语言
  • 隐藏谷歌翻译的样式，更美观
