神级桌面美化软件——SAO Utils

February 8, 2017
Ice-Hazymoon
阅读约 2 分钟
912 字
标签 桌面美化 软件
分类 萌化

今天在某群内看到了有人发这个软件的介绍，然后看了下使用视频，感觉真的好霸气，可怕的是这款软件在2015年就出了，然而我到现在才知道_(:з」∠)_

SAO Utils是一款由热风CG工作室开发的“SAO风格启动菜单”，高度还原了动漫刀剑神域中的操作方式。不过目前软件尚处于测试阶段。

软件总体使用下来感觉还不错，cpu、内存神马的占用也不是很高，感觉和雨滴相似，支持桌面挂件。听说雨滴和su+Wallpaper Engine更配哦。这软件还支持VR！支持VR！！支持VR！！！

视频介绍：http://www.bilibili.com/video/av2774939/

软件介绍地址：http://www.gpbeta.com/post/develop/sao-utils/

根据发布的下载地址下载以后安装完是酱紫的（ ( *￣▽￣)发现壁纸好像有点不配诶）

之后会让你输入用户名，弹出一个对话框，应该是在线使用者的互动吧。反正我弄了半天才关掉那个对话框。。。

关闭聊天窗口的三种方法：(首选项在任务栏，右键十字架)

  • ①首选项→插件→桌面网页挂件→禁用
  • ②在聊天窗口右键→显示挂件菜单→点加号（出来一个黄色的圈，这是网页挂件的初始图标）→再次在聊天窗口右键显示挂件菜单→点减号
  • ③下拉菜单→设置→[Dption]→插件→桌面网页→右键禁用、

桌面上还有个邮件收发的东西，也不错，支持收发邮件，大家可以去试试

最让我激动的是这软件里的浏览器，加上VR效果简直了~。预览图如下，你们自己感受_(:з」∠)_

还有实时追番功能

启动菜单也支持更换主题（SAO看烦了可以换成GGO，也有很多第三方主题~）

内存和CPU的占用也不是很高，不像别人说的那样会很卡，只要不是上世纪的电脑都能正常使用，毫无卡顿

其他功能就不介绍了，你们自己去下载体会吧，总体感觉很酷炫，而且支持插件和自定义~

下载地址： http://www.gpbeta.com/post/develop/sao-utils/
